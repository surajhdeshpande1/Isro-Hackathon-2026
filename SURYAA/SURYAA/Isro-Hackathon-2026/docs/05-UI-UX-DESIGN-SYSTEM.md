# TAPAS SURYA DRISHTI - Complete UI/UX Design System

## Core Design Philosophy

**The map is the product.**
**The city is the interface.**
**Climate intelligence is the content.**
**Decision-making is the outcome.**

The user should never feel they are navigating pages.
The user should feel they are operating a living city.

## Motion Design System

**Library:** Framer Motion

**Animation Types:**
- Map Flyovers (smooth camera transitions)
- Boundary Draw Animations (progressive reveal)
- District Reveal Animations (focus highlight)
- Heat Emergence Animations (gradual appearance)
- Scenario Morphing (transition between states)
- Financial Curve Animations (growth visualization)
- Roadmap Progressions (timeline playback)

**All animations are:**
- Smooth (easing curves)
- Elegant (never jarring)
- Premium (high polish)
- Purposeful (meaningful motion)
- 60 FPS (smooth performance)

## Microinteractions

Every interaction should respond instantly and feel alive:

- **Hover states:** Subtle highlight, color shift
- **Click feedback:** Brief scale pulse, visual confirmation
- **Search:** Live results, autocomplete, city suggestions
- **Layer toggle:** Smooth fade in/out
- **Scenario comparison:** Smooth morphing between states
- **District selection:** Highlight + data panel animation
- **Map navigation:** Smooth pan and zoom

## Performance Targets

- First Contentful Paint: < 1.5s
- Largest Contentful Paint: < 2.5s
- Cumulative Layout Shift: < 0.1
- Time to Interactive: < 3.5s
- Map load time: < 2s
- Hotspot discovery: < 5s

## Responsive Design

Mobile-first approach supporting:
- **Mobile** (320px - 640px)
- **Tablet** (641px - 1024px)
- **Desktop** (1025px - 1600px)
- **Ultra-wide** (1601px+)

Map always takes priority.
Intelligence panels adapt layout.
Typography scales responsively.
